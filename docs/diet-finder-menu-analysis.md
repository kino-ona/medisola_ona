# 메디솔라 식단 찾기 개선 프로젝트 - 메뉴 분석 자료

## 📋 프로젝트 개요

**목표:** 사용자가 25개 식단 중 자신에게 맞는 식단과 메뉴를 쉽게 찾을 수 있도록 퀴즈 기반 추천 시스템 구축

**핵심 요구사항:**
1. 건강 목표 + 재료 취향을 반영한 식단 추천
2. 식단 선택 후 개인 취향에 맞는 메뉴 필터링
3. 알러지/제외 재료 반영

**제외 사항:**
- 새벽배송 라인 (냉장): 프리미엄 영양케어, 프리미엄 당뇨케어 - 운영 종료 예정
