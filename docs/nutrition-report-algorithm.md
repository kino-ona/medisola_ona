# 영양 분석 결과 알고리즘 — 팀 검토용

> 최종 업데이트: 2026-03-23
> 소스 코드: `godo-dev-module/Component/DietFinder/DietFinder.php` → `generateNutritionReport()`
> 이 문서의 모든 테이블은 스프레드시트에 복사하여 검토/수정 후 개발팀에 전달할 수 있습니다.

---

## 1. 개요

사용자가 온보딩 퀴즈에서 입력한 **성별, 출생년도, 키, 체중, 건강목표/질환**을 기반으로 개인화 영양 분석 결과를 생성합니다. 리포트는 5개 섹션으로 구성됩니다:

```
① 코칭 메시지 → ② 상태 요약 카드 → ③ 핵심 영양 전략 → ④ 추천 영양 라인 → ⑤ 브랜드 메시지
```

현재는 **규칙 기반**으로 동작하며, 향후 AI API로 대체 가능하도록 설계되어 있습니다.

---

## 2. 입력 데이터

사용자가 퀴즈에서 입력하는 항목:

| 항목 | 형식 | 범위 | 예시 |
|------|------|------|------|
| 성별 | `male` / `female` | 2개 | `male` |
| 출생년도 | 정수 | 1926~2006 | `1990` |
| 키 | 소수점 (cm) | 120~220 | `175.0` |
| 체중 | 소수점 (kg) | 30~150 | `70.5` |
| 건강목표/질환 | 문자열 배열 | 아래 표 참조 | `["weight-loss"]` 또는 `["diabetes"]` |

### 2-1. 건강 목표 (단일 선택)

7개 중 1개만 선택할 수 있습니다. "질환 관리"를 선택하면 하위 질환을 복수 선택합니다.

| ID | 화면 표시명 | 부제 | 색상 | 아이콘 |
|----|-----------|------|------|--------|
| `weight-loss` | 체중 감량 | 건강한 다이어트 | #22C55E | trendingDown |
| `general` | 균형잡힌 식단 | 전반적 건강 관리 | #10B981 | smile |
| `weight-gain` | 체중 증량 | 건강한 체중 증가 | #F97316 | trendingUp |
| `muscle-gain` | 근육 증량 | 고단백 식단 | #EF4444 | dumbbell |
| `pregnancy` | 임신·수유부 관리 | 엄마와 아기를 위한 영양 | #EC4899 | baby |
| `growth` | 성장 관리 | 성장기 맞춤 영양 | #8B5CF6 | sprout |
| `aging` | 노화 관리 | 건강한 노년을 위한 영양 | #6366F1 | shield |

### 2-2. 질환 관리 (복수 선택)

건강 목표에서 "질환 관리"를 선택한 경우에만 노출됩니다. 복수 선택 가능.

| ID | 화면 표시명 | 부제 | 색상 | 비고 |
|----|-----------|------|------|------|
| `diabetes` | 당뇨 | 혈당 관리가 필요한 분 | #3B82F6 | |
| `gestational-diabetes` | 임신성 당뇨 | 임신 중 혈당 관리 | #EC4899 | 여성 전용 |
| `kidney` | 신장 질환 | 신장 기능 보호가 필요한 분 | #8B5CF6 | 하위옵션 존재 |
| `kidney-pre-dialysis` | ∟ 투석 전 단계 | 신장 기능 보존 식단 | #8B5CF6 | kidney 선택 시 표시 |
| `kidney-dialysis` | ∟ 투석 중 | 투석 환자 맞춤 식단 | #8B5CF6 | kidney 선택 시 표시 |
| `cancer` | 암 투병 | 영양 보충이 필요한 분 | #F472B6 | |

### 2-3. 프로필 기반 선택 제약

사용자의 신체 계측 정보에 따라 상충되는 목표/질환은 **비활성**(선택 불가)됩니다.

| 대상 | 제약 조건 | 사유 메시지 |
|------|----------|------------|
| 체중 감량 | BMI < 18.5 (저체중) | 저체중 상태에서는 체중 감량이 권장되지 않습니다 |
| 체중 증량 | BMI ≥ 25 (비만) | 비만 상태에서는 체중 증량이 권장되지 않습니다 |
| 임신·수유부 관리 | 남성 | 여성 전용 목표입니다 |
| 성장 관리 | 만 25세 이상 | 성장기(만 24세 이하) 대상 목표입니다 |
| 노화 관리 | 만 40세 미만 | 만 40세 이상 대상 목표입니다 |
| 임신성 당뇨 (질환) | 남성 | 여성 전용 질환입니다 |

### 2-4. 선택 규칙 요약

```
건강 목표: 단일 선택 (7개 중 1개)
  ├── 일반 목표 선택 → 질환 섹션 닫힘, 질환 선택 초기화
  └── "질환 관리" 선택 → 질환 섹션 열림, 질환 1개 이상 필수

백엔드 전달: goal + diseases 병합 (disease-management ID는 제외)
  예: 체중 감량 → ["weight-loss"]
  예: 질환 관리 + 당뇨 → ["diabetes"]
```

## 3. BMI 계산

### 3-1. BMI 값

```
BMI = 체중(kg) / (키(m))²
예: 70kg / (1.75m)² = 22.9
```

### 3-2. BMI 상태 분류 (한국 기준)

| BMI 범위 | 상태 | 상태키 | 시각화 색상 |
|----------|------|--------|------------|
| < 18.5 | 저체중 | `underweight` | #3B82F6 (파란색) |
| 18.5 ~ 22.9 | 정상 | `normal` | #10B981 (녹색) |
| 23.0 ~ 24.9 | 과체중 | `overweight` | #FBBF24 (노란색) |
| ≥ 25.0 | 비만 | `obese` | #EF4444 (빨간색) |

> WHO 기준(25 이상=과체중)과 다르게, 한국 비만학회 기준(23 이상=과체중)을 적용합니다.

### 3-3. 일일 필요 칼로리 (Mifflin-St Jeor 공식)

**기초대사량(BMR) 계산:**

| 성별 | 공식 |
|------|------|
| 남성 | BMR = (10 × 체중kg) + (6.25 × 키cm) - (5 × 나이) + 5 |
| 여성 | BMR = (10 × 체중kg) + (6.25 × 키cm) - (5 × 나이) - 161 |

**일일 필요 칼로리:**

```
일일칼로리 = BMR × PAL(신체활동계수)
```

**PAL(Physical Activity Level) 분기:**

| 구분 | PAL | 기준 |
|------|-----|------|
| 비질환자 (건강 목표만 선택) | 1.375 | 가벼운 활동 (주 1~3회 운동) |
| 질환자 (질환 관리 목표 포함) | 1.2 | 저활동 (안정 시) |

**질환 조건 ID:**
`diabetes`, `gestational-diabetes`, `kidney-pre-dialysis`, `kidney-dialysis`, `cancer`

> 위 질환 ID 중 하나라도 선택되어 있으면 PAL 1.2 적용. 건강 목표(`weight-loss`, `muscle-gain`, `general` 등)만 선택한 경우 PAL 1.375 적용.

### 3-4. 일일 단백질 목표

```
단백질(g) = 체중(kg) × 1.4
예: 70kg × 1.4 = 98g/일
```

> 계수 1.4는 일반 성인 활동량 기준. 근육 증량 목표 시 1.6~2.0으로 조정 검토 가능.

### 계산 예시

**비질환자 (PAL 1.375):**

| 항목 | 남성 35세 175cm 70kg | 여성 50세 160cm 55kg |
|------|---------------------|---------------------|
| BMI | 22.9 (정상) | 21.5 (정상) |
| BMR | 1683 kcal | 1193 kcal |
| 일일칼로리 (BMR × 1.375) | 2300 kcal | 1600 kcal |
| 단백질 목표 | 98g | 77g |

**질환자 (PAL 1.2):**

| 항목 | 남성 35세 175cm 70kg (당뇨) | 여성 50세 160cm 55kg (신장질환) |
|------|---------------------------|-------------------------------|
| BMI | 22.9 (정상) | 21.5 (정상) |
| BMR | 1683 kcal | 1193 kcal |
| 일일칼로리 | 2020 kcal | 1432 kcal |
| 단백질 목표 | 98g | 77g |

## 4. 코칭 메시지 (① 섹션)

리포트 최상단에 **1~3개**의 메시지 단락을 표시합니다. 메시지는 아래 5가지 소스에서 순서대로 조립됩니다. (현재 4-2, 4-3은 임시 숨김 상태)

### 4-1. 기본 소개 + 건강 목표 (고정 — 항상 1개)

```
"{성별}, 만 {나이}세, {키}cm / {체중}kg 기준으로 분석한 결과입니다.
현재 BMI는 {BMI값}로 {BMI상태} 범위에 해당하며, {목표/질환 라벨}을(를) 목표로 하고 있습니다."
```

**목표 라벨 조립 규칙:**
- 건강 목표 + 질환을 쉼표로 연결
- 을/를 조사는 마지막 글자의 받침 유무로 자동 결정
- 예: `체중 감량을 목표로` / `당뇨 관리, 고콜레스테롤 관리를 목표로` / `근육 증량을 목표로`

| 조건 ID | 라벨 표시 |
|---------|----------|
| `weight-loss` | 체중 감량 |
| `general` | 균형잡힌 식단 |
| `weight-gain` | 체중 증량 |
| `muscle-gain` | 근육 증량 |
| `pregnancy` | 임신·수유부 관리 |
| `growth` | 성장 관리 |
| `aging` | 노화 관리 |
| `diabetes` | 당뇨 관리 |
| `gestational-diabetes` | 임신성 당뇨 관리 |
| `kidney-pre-dialysis` | 신장 질환(투석 전) 관리 |
| `kidney-dialysis` | 신장 질환(투석 중) 관리 |
| `cancer` | 암 투병 영양 관리 |

### 4-2. 연령대별 조언 — ⚠️ 임시 숨김 (TEMPORARILY HIDDEN)

> **현재 코드에서 주석 처리되어 표시되지 않습니다.** 향후 재활성화 시 아래 내용이 적용됩니다.

| 나이 | 메시지 |
|------|--------|
| 20대 (< 30세) | 20대는 활동량이 많아 에너지 소비가 높은 시기입니다. 이 시기에 올바른 식습관을 형성하면 30대 이후 건강 관리에 큰 도움이 됩니다. |
| 30대 (30~39세) | 30대부터는 기초대사량이 매년 1~2%씩 감소합니다. 근손실 방지와 대사 유지를 위한 단백질 섭취가 중요합니다. |
| 40대 (40~49세) | 40대는 심혈관 건강에 주의가 필요한 시기입니다. 나트륨과 포화지방 조절, 오메가3 섭취가 핵심입니다. |
| 50대 (50~59세) | 50대는 골밀도 감소와 근감소증 예방이 중요합니다. 충분한 단백질과 칼슘, 비타민D 섭취를 권장합니다. |
| 60대 이상 (≥ 60세) | 60대 이상은 소화 기능과 영양 흡수율이 저하되는 시기입니다. 소화가 잘되는 양질의 단백질과 충분한 수분 섭취가 필요합니다. |

### 4-3. 성별 × 연령대 조언 — ⚠️ 임시 숨김 (TEMPORARILY HIDDEN)

> **현재 코드에서 주석 처리되어 표시되지 않습니다.** 향후 재활성화 시 아래 내용이 적용됩니다.

| 조건 | 메시지 |
|------|--------|
| 여성 + 50세 미만 | 여성은 월경주기에 따른 철분 손실에 주의가 필요합니다. 철분이 풍부한 식품과 함께 비타민C를 섭취하면 흡수율이 높아집니다. |
| 여성 + 50세 이상 | 폐경 이후에는 에스트로겐 감소로 골다공증 위험이 높아집니다. 칼슘과 비타민D, 단백질 섭취를 강화해야 합니다. |
| 남성 + 40세 이상 | 40대 이후 남성은 복부 지방 축적과 전립선 건강에 주의가 필요합니다. 식이섬유와 항산화 영양소 섭취를 늘리세요. |
| 남성 + 40세 미만 | 남성은 근육량 유지를 위해 체중 kg당 1.2~1.6g의 단백질 섭취를 권장합니다. |

### 4-4. 건강 목표별 맞춤 코칭 (조건부 — 최대 1개)

사용자가 선택한 **첫 번째 건강 목표**에 해당하는 메시지 1개를 추가합니다. `<br><br>`로 단락을 구분하며, 하나의 `<p>` 태그 안에 렌더링됩니다.

| 목표 ID | 코칭 메시지 |
|---------|------------|
| `weight-loss` | 고객님의 목표인 '체중 감량'을 위해 충분한 단백질 섭취로 근손실을 방지하면서, 지방과 당류 섭취를 적정 수준으로 조절하는 것이 중요합니다. 또한 운동 및 식이요법을 병행하여 평소 섭취량보다 하루 약 500kcal를 줄이면, 주당 약 0.5kg, 한달에 약 2kg의 체중 감량을 기대할 수 있습니다. 이러한 점진적인 체중 감량 방법이 권장됩니다.`<br><br>`고객님의 키, 체중, 목표 등을 고려 시, 하루 약 **{dailyCalories}kcal**, 단백질 약 **{proteinG}g** 권장드립니다. 이를 바탕으로 아래와 같이 메디쏠라 식단 안내드립니다. |
| `weight-gain` | 고객님의 목표인 체중 증량을 위해서는 충분한 열량을 섭취하고, 단백질을 체중 1kg당 약 1.2–2.0g 수준으로 보충하여 근육 증가를 돕는 것이 중요합니다. 또한 견과류, 올리브유 등 건강한 지방원을 활용하여 에너지 섭취를 보충하는 것이 도움이 됩니다. 평소 섭취량보다 하루 약 250–500 kcal를 추가로 섭취하고 운동을 병행하여 주당 약 0.2–0.5 kg의 점진적인 체중 증가를 목표로 하는 것이 권장됩니다. |
| `muscle-gain` | 고객님의 목표인 '근육 증량'을 위해 체중 kg당 1.3~2.0g의 단백질을 섭취하고, 충분한 칼로리를 확보하는 것이 중요합니다. |
| `general` | 고객님의 목표인 '균형잡힌 식단'을 위해 탄수화물, 단백질, 지방의 균형 잡힌 섭취와 충분한 식이섬유, 오메가3를 챙기는 것이 중요합니다. |
| `pregnancy` | 고객님의 목표인 '임신·수유부 관리'를 위해 양질의 단백질과 DHA/EPA, 엽산, 철분을 충분히 섭취하는 것이 중요합니다. |
| `growth` | 고객님의 목표인 '성장 관리'를 위해 충분한 칼로리와 양질의 단백질, 칼슘, 오메가3를 균형 있게 섭취하는 것이 중요합니다. |
| `aging` | 고객님의 목표인 '노화 관리'를 위해 근감소증 예방을 위한 단백질과 혈관 건강을 위한 오메가3, 소화 건강을 위한 식이섬유를 충분히 섭취하는 것이 중요합니다. |

> 질환 관리를 선택한 경우(건강 목표 = disease-management), 이 섹션은 건너뜁니다. 대신 4-5 질환별 코칭이 적용됩니다.
>
> **동적 변수 (weight-loss만 해당):**
> - `{dailyCalories}`: 섹션 3-3에서 계산된 일일 필요 칼로리 (정수, 반올림)
> - `{proteinG}`: 섹션 3-4에서 계산된 일일 단백질 목표 (정수, 반올림)
> - `<strong>` 태그로 볼드 강조 렌더링

### 4-5. 질환별 맞춤 코칭 (조건부 — 질환당 1개씩)

**diabetes (당뇨)**
> 당뇨는 안정적인 혈당관리가 필요합니다. 아래와 같은 식사관리 목표가 필요합니다.
> 1. 안정적인 혈당관리
> 2. 단순당 제한
>
> 안정적인 혈당관리를 위해 식사는 일정한 시간에 맞춰 규칙적으로 섭취하고, 간식은 식후 2~3시간 간격을 두고 섭취하는 것이 중요합니다. 또한, 사탕, 초콜릿, 가당음료와 같은 단순당 함량이 높은 식품은 제한하는 것이 좋습니다.
> 이를 기반으로 개인의 혈당 조절 상태에 맞는 하루 권장 열량과 탄수화물 섭취량, 단백질 필요량을 설정하고, 혈당 변동을 최소화하는 식단 구성이 필요합니다.

**gestational-diabetes (임신성 당뇨)**
> 임신성 당뇨는 태아와 산모 모두를 위해 안정적인 혈당 유지가 중요합니다. 아래와 같은 식사관리 목표가 필요합니다.
> 1. 식후 혈당 상승 최소화
> 2. 균형 잡힌 탄수화물 분배
>
> 임신 중 안정적인 혈당관리를 위해 하루 3끼 식사와 2~3회의 간식을 소량씩 나누어 섭취하는 것이 중요합니다. 특히 아침에는 혈당 상승이 크므로 아침 식사 시 탄수화물 섭취량을 조절해 주세요. 또한, 설탕, 주스 등 혈당을 빠르게 올리는 식품은 제한하고, 통곡물, 채소 등 식이섬유가 풍부한 식품을 선택하는 것이 좋습니다.
> 이러한 기준을 바탕으로 개인의 상태에 맞는 하루 권장 열량과 탄수화물 분배, 단백질 필요량을 설정하고 이에 맞는 식단 구성이 필요합니다.

**kidney-pre-dialysis (신장질환 — 투석 전)**
> 신장질환(투석 전)은 신장 기능 저하를 늦추는 것이 중요합니다. 아래와 같은 식사관리 목표가 필요합니다.
> 1. 단백질 적정 섭취
> 2. 나트륨 제한
>
> 신장 부담을 줄이기 위해 단백질 섭취는 과도하지 않게 조절하며, 가공식품이나 국물류 섭취를 줄여 나트륨 섭취를 제한하는 것이 중요합니다. 또한, 개인 상태에 따라 칼륨과 인 섭취 조절이 필요할 수 있으므로 식품 선택에 주의가 필요합니다.
> 이에 따라 신장 기능 단계에 맞는 하루 단백질 제한량과 열량 기준을 설정하고, 전해질 균형을 고려한 식단 구성이 필요합니다.

**kidney-dialysis (투석 중)**
> 투석 중에는 영양 손실이 증가하므로 충분한 영양 공급이 중요합니다. 아래와 같은 식사관리 목표가 필요합니다.
> 1. 충분한 단백질 섭취
> 2. 수분 및 전해질 조절
>
> 투석으로 손실되는 단백질을 보충하기 위해 양질의 단백질(육류, 생선, 계란 등)을 충분히 섭취하는 것이 중요합니다. 또한, 체액 과다를 방지하기 위해 수분 섭취를 조절하고, 칼륨과 인이 높은 식품은 제한이 필요합니다.
> 이를 기반으로 투석 환자에 맞는 증가된 단백질 요구량과 열량 기준을 반영하여 식단을 구성하는 것이 필요합니다.

**cancer (암 투병)**
> 암 환자는 질환 자체와 항암치료로 인해 식욕 저하, 염증 반응 증가, 대사 변화 등이 발생하여 영양불량과 근육 손실이 쉽게 나타날 수 있습니다. 이에 따라 다음과 같은 식사관리 목표가 필요합니다.
> 1. 체중 및 근육량 유지
> 2. 충분한 에너지 및 단백질 섭취
>
> 영양불량을 예방하기 위해 식사는 가능한 범위 내에서 자주 나누어 섭취하고, 특히 단백질이 풍부한 식품을 충분히 섭취하는 것이 중요합니다. 이는 근육 감소를 예방하고 치료 과정에서의 회복과 예후 개선에 도움을 줄 수 있습니다.
> 이를 기반으로 영양불량 예방과 근육량 유지를 위한 충분한 열량과 단백질 요구량을 설정하고, 개인의 섭취 가능 상태를 반영한 식단 구성이 필요합니다.

## 5. 하루 필요 영양소 (② 섹션)

BMI 계산에서 산출된 일일 필요 칼로리를 기반으로 3대 매크로 영양소와 미시 영양소 목표를 표시합니다.

### 5-1. 매크로 영양소 배분

| 영양소 | 계산 공식 | 색상 |
|--------|----------|------|
| 단백질 | 체중(kg) × 1.4 → g, 나머지 비율 자동 | #EF4444 (빨간색) |
| 지방 | 일일칼로리 × 30% / 9 → g | #F59E0B (노란색) |
| 탄수화물 | (일일칼로리 − 단백질kcal − 지방kcal) / 4 → g | #3B82F6 (파란색) |

**비율 계산 흐름:**
```
proteinCal = proteinG × 4
proteinPct = proteinCal / dailyCalories × 100
fatPct = 30 (고정)
carbsPct = 100 − proteinPct − fatPct
```

**예시 (남성 35세, 70kg, 일일 2609kcal):**

| 영양소 | g | kcal | 비율 |
|--------|---|------|------|
| 탄수화물 | 364g | 1456 | 56% |
| 단백질 | 98g | 392 | 15% |
| 지방 | 87g | 783 | 30% |

### 5-2. 미시 영양소 목표

| 항목 | 기본값 | 조건부 조정 | 아이콘 | 색상 |
|------|--------|-----------|--------|------|
| 당류 섭취 | 25g 이하 | — | trendingDown | #8B5CF6 |
| 나트륨 섭취 | 2000mg 이하 | 신장 질환 → 1500mg | shield | #3B82F6 |
| 오메가-3 지방산 | 500mg 이상 | — | fish | #0EA5E9 |

> **검토 포인트**: 나트륨 제한은 신장 질환(`kidney-pre-dialysis`, `kidney-dialysis`) 선택 시 1500mg으로 강화됩니다.

## 6. 핵심 영양 전략 (③ 섹션) — 통합 알고리즘

> - `protein-up`의 목표값 = 체중(kg) × 1.4. 예: 70kg → "최소 98g/일"

| `muscle-gain` | 근육 증량 | `protein-up` | 근육 합성에 필요한 양질의 단백질을 충분히 공급합니다. | `calorie-up` | 근육 성장에 필요한 충분한 칼로리를 확보합니다. | `omega3-up` | 항염 효과로 운동 후 회복을 돕고 근육 성장을 지원합니다. |
| `pregnancy` | 임신·수유부 관리 | `omega3-up` | 태아 두뇌 발달과 모체의 혈관 건강을 위해 DHA/EPA를 보충합니다. | `protein-up` | 태아 발달과 모체 건강 유지를 위해 양질의 단백질을 공급합니다. | `calorie-up` | 태아 성장과 모체 건강 유지를 위해 충분한 칼로리를 확보합니다. |
| `growth` | 성장 관리 | `calorie-up` | 성장기에 필요한 충분한 에너지를 확보합니다. | `protein-up` | 뼈와 근육 성장에 필수적인 단백질을 충분히 공급합니다. | `omega3-up` | 두뇌 발달과 집중력 향상을 위해 오메가3를 보충합니다. |
| `aging` | 노화 관리 | `omega3-up` | 혈관 건강과 인지 기능 유지를 위해 오메가3를 보충합니다. | `protein-up` | 근감소증 예방과 체력 유지를 위해 양질의 단백질을 공급합니다. | `sugar-down` | 노화에 따른 대사 건강 유지를 위해 당류 섭취를 조절합니다. |

> **참고**: `muscle-gain`은 4순위로 `saturated-fat-down`이 매핑에 포함되어 있습니다. 단독 선택 시 최대 3개 제한(`array_slice`)으로 표시되지 않지만, 다른 조건과 조합하여 1~3순위 전략이 이미 추가된 경우 `saturated-fat-down`이 추천될 수 있습니다.

## 7. 추천 영양 라인 (④ 섹션) — 전략 연동

### 7-1. 영양 라인 데이터베이스

각 라인의 고정 속성입니다. 전략 카드가 선택되면, 그에 연결된 라인이 자동으로 추천됩니다. `reason`은 6-2 매핑 테이블의 "추천 사유"가 우선 적용됩니다.

| 라인 ID | 표시명 | 색상 | 기본 설명 (reason) | 혜택 태그 (benefits) |
|---------|--------|------|-------------------|---------------------|
| `high-protein` | 고단백 라인 | #EF4444 | 양질의 단백질로 근손실을 방지하고 체력을 유지합니다. | 근육 보호, 포만감 유지, 기초대사량 향상 |
| `low-sodium` | 저나트륨 라인 | #3B82F6 | 저염식으로 혈압과 신장 건강을 보호합니다. | 혈압 관리, 부종 감소, 신장 보호 |
| `omega3` | Omega-3 풍부 | #0EA5E9 | 오메가3 지방산으로 혈관 건강과 염증 조절을 돕습니다. | 혈관 건강, 항염 효과, 두뇌 기능 |
| `cal-550` | 550kcal 라인 | #F97316 | 충분한 칼로리 공급으로 활동량이 많거나 성장기에 필요한 에너지를 확보합니다. | 에너지 공급, 근육 성장 지원, 활동량 보충 |
| `low-sugar` | 저당 라인 | #8B5CF6 | 당류를 최소화하여 혈당 변동을 줄입니다. | 혈당 안정, 인슐린 감수성, 에너지 안정 |
| `low-saturated-fat` | 저포화지방 라인 | #F59E0B | 포화지방 섭취를 줄여 심혈관 건강을 돕습니다. | 심장 건강, 체지방 감소, 소화 개선 |
| `low-cholesterol` | 저콜레스테롤 라인 | #EC4899 | 콜레스테롤 섭취를 제한하여 혈중 수치를 관리합니다. | 혈관 건강, LDL 감소, 심장 보호 |

> **참고**: 440kcal 라인은 기본값이므로 리포트에서 별도 추천하지 않습니다. 고칼로리가 필요한 경우에만 550kcal 라인을 추천합니다.

### 7-2. 통합 결정 로직

전략 카드와 추천 라인이 **하나의 루프**에서 동시에 결정됩니다.

```
1. 사용자가 선택한 모든 조건을 순서대로 처리
2. 각 조건의 매핑(6-2 테이블)에서 전략 ID를 순서대로 추가
3. 이미 추가된 전략 ID는 건너뜀 (중복 방지, 먼저 추가된 것이 우선)
4. 전략이 추가될 때, 해당 전략의 "연결 라인"도 함께 추가 → 추천 라인 결정
5. 어떤 조건에서도 omega3-up이 추가되지 않았고 전략이 3개 미만이면, 자동 추가
6. 전략 카드와 추천 라인 모두 최대 3개로 제한
```

**핵심: 전략 카드 ↔ 추천 라인이 항상 1:1 대응**
- 전략 카드에 "단백질 섭취 강화"가 표시되면, 추천 라인에 "고단백 라인"이 반드시 표시
- 전략이 3개면 추천 라인도 3개 (동일 순서)

## 12. 향후 AI API 전환 시 고려사항

현재 `generateNutritionReport()`는 규칙 기반입니다. AI API로 전환할 경우:

1. **입력은 동일**: `{ gender, birthYear, height, weight, conditions }` — 변경 불필요
